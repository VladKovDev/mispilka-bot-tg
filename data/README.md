# Папка data

Эта папка содержит данные работы бота. Все файлы формата JSON.

## Файлы

### messages.json
Настройка очереди сообщений и их параметров.

Структура:
```json
{
  "messages_list": ["msg1", "msg2", "msg3"],
  "messages": {
    "msg1": {
      "timing": [часы, минуты],
      "url_button": ["ссылка", "текст_кнопки"]
    }
  }
}
```

- `messages_list` - очередь сообщений (обрабатывается с конца)
- `timing` - задержка перед отправкой следующего сообщения `[часы, минуты]`
- `url_button` - инлайн-кнопка со ссылкой (можно оставить пустые строки)

### messages/*.md
Файлы с текстом сообщений. Иена файлов должны соответствовать ключам из `messages.json`.

Пример: `data/messages/start.md`

### users.json
База пользователей. Автоматически заполняется ботом.

### schedule_backup.json
Бэкап расписания. Автоматически создается и используется при рестарте бота.

### commands.json
Конфигурация команд (в разработке).

## Как использовать

1. Создайте файлы с текстом сообщений в `data/messages/`
2. Настройте `messages.json` - добавьте сообщения в очередь
3. Укажите `timing` для каждого сообщения (через сколько часов/минут после предыдущего)
4. При необходимости добавьте URL-кнопки
5. Бот автоматически будет отправлять сообщения из очереди
