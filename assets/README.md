# Папка assets

Эта папка содержит статические файлы, которые бот отправляет пользователям.

## Структура

### images/
Изображения для сообщений. Названия файлов должны соответствовать именам сообщений из `data/messages.json`.

**Формат:** PNG
**Шаблон имени:** `{имя_сообщения}.PNG`

Пример: если в `messages.json` есть сообщение `welcome`, здесь должен быть файл `welcome.PNG`.

**Примечание:** Эта папка не хранится в репозитории. Файлы изображений добавляются на сервере вручную.

## Использование

1. Настройте сообщения в `data/messages.json`, указав имена картинок
2. Добавьте изображения в папку `images/` на сервере с правильными именами
3. Бот автоматически прикрепит изображение к сообщению при отправке
