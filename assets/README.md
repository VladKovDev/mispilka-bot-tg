# Папка assets

Эта папка содержит статические файлы, которые бот отправляет пользователям.

## Структура

### documents/
Документы, отправляемые ботом:

- `ОФЕРТА.docx` - Оферта (договор)
- `Политика конфиденциальности.docx` - Политика конфиденциальности
- `Согласие ТГ БОТ.docx` - Согласие на использование Telegram бота

### images/
Изображения для сообщений. Названия файлов должны соответствовать именам сообщений из `data/messages.json`.

**Формат:** PNG
**Шаблон имени:** `{имя_сообщения}.PNG`

Пример: если в `messages.json` есть сообщение `welcome`, здесь должен быть файл `welcome.PNG`.

## Использование

1. Положите документы в папку `documents/`
2. Положите изображения в папку `images/` с правильными именами
3. Настройте сообщения в `data/messages.json`, указав имена картинок
4. Бот автоматически прикрепит изображение к сообщению при отправке
